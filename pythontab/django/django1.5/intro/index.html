如果你是编程新手，那么你最好先看看 编程新手 Python 资源列表

如果你有其他语言的编程经验并想快速学习 Python ，那么我们推荐 Dive Into Python (有 纸质书 和 中文版). 如果这本书不符合你的要求，这里还有许多其他 关于 Python 的书.
