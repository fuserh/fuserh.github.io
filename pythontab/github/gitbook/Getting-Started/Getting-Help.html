获取帮助¶

想了解 Git 的各式工具该怎么用，可以阅读它们的使用帮助，方法有三:

$ git help <verb>
$ git <verb> --help
$ man git-<verb>

比如，要学习 config 命令可以怎么用，运行:

$ git help config

我们随时都可以浏览这些帮助信息而无需连网。不过，要是你觉得还不够，可以到 Frenode IRC 服务器（irc.freenode.net）上的 #git 或 #github 频道寻求他人帮助。这两个频道上总有着上百号人，大多都有着丰富的 git 知识，并且乐于助人。
